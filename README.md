## 💡 О проекте

Искра — это веб-сайт психологических услуг, разработанный на React.
Главная идея проекта — сделать психологию более доступной и понятной через визуальные образы эмоций и мягкий пользовательский опыт.

В качестве персонажей используются эмоции:
- 😊 Радость  
- 😡 Гнев  
- 😨 Страх  
- 😰 Тревожность  
- 😢 Грусть  
- 😶 Скука  

Каждая эмоция помогает пользователю лучше понять своё состояние и выбрать подходящего специалиста.

---

## Основной функционал

- Тестирование для определения подходящего психолога  
- Форма записи на консультацию  
- Каталог специалистов  
- Раздел со статьями  
- Карусели и интерактивные блоки  
- Анимации интерфейса  
- Интерактивные UI-элементы  
- Плавная навигация по секциям
